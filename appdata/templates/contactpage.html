Game Rules

A player can join a randomly generated game, that is open to everyone up to capacity (20 people). The games, that are about to start, will be shown on the main page. From there the user will be able to check the bingo card, and join the game, if he feels like it. Or if the player is a member of a team, that team can create a game that only this group of users will play. The team is created by one user, the team moderator, who can add other users to the team. Or a user can ask to join the team, in which case this request can be approved by team moderator. The moderator can create a game, and choose if the game is open to public or not. If the game is created as public, anybody can join it, and it will be shown on the main page and on the games page. A player is then presented with the bingo card. Each cell in the bingo card represents one activity. To fill the bingo card, a player must follow the instructions and fulfil the activity, and then check the activity as complete, using the given passcode or QR-code. First player, who fills all the cells in the bingo card wins! The game can have a time limit, or not. One game can be played by up to 20 people. One person can also play the game casually, and use the bingo card as a suggestion for his daily activities. Some of the activities will have a timeframe, depending on the venue work hours.
